spun

Deploy SSR apps with one command. Live in seconds, expires in 24 hours.

Install
npm i -g spunrun
Deploy
spun deploy

Run from any SSR project directory. No config needed.

That's it
https://your-app.spun.run
Other commands
  • spun ls — list your apps
  • spun logs <name> — view logs
  • spun rm <name> — remove an app
